# 🟢 ----- Intelligent Candidate Evaluation and Interview Assist -----

* “This project had three components: resume summarization, interview question generation, and candidate screening.”
* “For summarization, we generated concise candidate insights using LLMs and refined them using recruiter feedback.”
* “For interview questions, we created personalized, role-aligned questions using resume and JD, including support for 1-way video interviews.”
* “For screening, we used a dual approach—matching candidates against an ideal profile derived from historical hires, and fallback JD-based evaluation.”
* “Technically, we designed it as a pipeline with structured outputs, and validated it using recruiter feedback and historical data to ensure consistency and reliability.”
