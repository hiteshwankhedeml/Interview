# 🟢 Elastic Vector DB

* Vector search capability built into Elasticsearch 8+
* <mark style="color:purple;background-color:purple;">**Uses**</mark><mark style="color:purple;background-color:purple;">**&#x20;**</mark><mark style="color:purple;background-color:purple;">**`dense_vector`**</mark><mark style="color:purple;background-color:purple;">**&#x20;**</mark><mark style="color:purple;background-color:purple;">**field type**</mark>
* Supports ANN search using HNSW algorithm
* <mark style="color:purple;background-color:purple;">**Supports cosine, dot product, and L2 similarity**</mark>
* <mark style="color:purple;background-color:purple;">**Enables semantic search for RAG systems**</mark>
* <mark style="color:purple;background-color:purple;">**Supports hybrid search (BM25 + vector)**</mark>
* <mark style="color:purple;background-color:purple;">**Allows metadata filtering with vector queries**</mark>
* Distributed and horizontally scalable
* Supports sharding and replication
* Provides RBAC and enterprise security controls
* Offers REST APIs for integration
* Available as cloud-managed or self-hosted
* Suitable for enterprise-grade production deployments
