# 🟢 Document Verification - Malaysia

* MyKad (NRIC) — compulsory smart chip identification card for Malaysian citizens aged 12 and above
* Passport (for foreign candidates)
* Degree certificates
* Experience letters
* Offer / appointment letters



* Numbers have to match mandatorily ⇒ No LLM Call
* If numbers match then we pass to LLM with a prompt to verify the data as a document verification agent will do
