# 🟢 Communication Layer

* <mark style="color:purple;background-color:purple;">**Role: Facilitator between Generation and Validation outputs + reflections.**</mark>
*   <mark style="color:purple;background-color:purple;">**System prompt (summary)**</mark>

    * <mark style="color:purple;background-color:purple;">**Persona: Collaboration facilitator**</mark>
    * <mark style="color:purple;background-color:purple;">**Goals: align views, surface agree/disagree, consensus, action items**</mark>

    Input

    * Generation output + validation output
    * Both self-reflections
    * Iteration count
    * User requirements (as context)

    Output

    * `discussion` — facilitator narrative
    * `consensus` — lines matching keywords (_agree_, _both_, _aligned_, …)
    * `action_items` — lines matching _should_, _need to_, _improve_, …
