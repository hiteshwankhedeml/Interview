# Complete vs Chat Completion

* <mark style="color:purple;background-color:purple;">**Completion API**</mark>
  * <mark style="color:purple;background-color:purple;">**Takes a single prompt string as input.**</mark>
  * Stateless.
  * No explicit roles.
  * Best for simple text generation or transformations.
* <mark style="color:purple;background-color:purple;">**Chat Completion API**</mark>
  * <mark style="color:purple;background-color:purple;">**Takes a list of messages.**</mark>
  * Role-based (`system`, `user`, `assistant`).
  * Maintains conversational context.
  * Best for chatbots and multi-turn interactions.
